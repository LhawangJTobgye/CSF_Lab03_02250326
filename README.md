# Aim of The Lab #
To understand and learn how to use control structures in python by implementing conditional statements, loops, break and also continue statements. Also to develop and learn how to use functions for solving programming solutions in a efficient and structured manner.

# List of Programs #
1. Conditional Statements
2. for Loops and while loops
3. Break and Continue
4. Functions
5. Integrated Programs

# Explanation of each task #
Part_A: 
1. Write code to check positive, negative, or zero.
2. Write code to check Even or Odd.
3. Write code to find largest of two numbers.
4. Write code to find largest of three numbers using nested if-else.
5. Write a code to find the grade from marks.
6. Write code to create menu-driven calculator.

Part_B:
7. Write code to print numbers from 1 to 10 using for loop.
8. Write code to print only even numbers from 1 to 20.
9. Write a code to find the sum of first 10 natural numbers.
10. Write code to create multiplication table for any number.
11. Write code to print numbers from 1 to 5 using while loops.
12. Write code to find sum 1 to n using while loops.
13. Write code to keep taking inputs until 0.

Part_C:
14. Write code to print 1 to 10 but stop at 6 using break method.
15. Write code to print 1 to 10 but skip 5 using the continue method.
16. Write code to search a number in list using break method.

Part_D:
17. Write a code using functions to print output "Hello, World".
18. Write a code to create a function to return the square of a number.
19. Write a code to create a function to add two numbers.
20. Write a code to create a function to check if a number is odd or even.
21. Write a code to create a function that will return the factorial of a number.
22. Write a code to create a function that will print multiplication table of a number.

Part_E:
23. Write a code to create a function as well as use loops to print q to n and check if it is odd or even.
24. Write a code to create a menu-driven calculator using functions and loops.
